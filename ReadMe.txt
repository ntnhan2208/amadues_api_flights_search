Demo xây dựng một form tìm kiếm bằng boostrap để có thể nhập cái tiêu chí tìm kiếm chuyến bay.
Demo sử dụng một số phần quan trọng như: Amadeus Node SDK, Express, dotenv
Demo dùng để tìm thông tin các chuyến bay, qua đó tạo tiền đề cho việc xem giá, đặt vé máy bay thông qua API.
Để chạy được demo cần mở Terminal lên và chạy:
	npm install
	npm run start
